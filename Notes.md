Drawing On canvas
SVG -dom based but utilizes scalable Vector Graphics - focuses on shapes rather than text- embed directly in HTML document or use <img> tag

Canvas -single dom element that encapsulates a picture - it provides programming interface for drawing shapes onto space

Difference

The orginal description of the shapes is preserved so that it can be resized anytime in SVG
while in Canvas - it converts the shapes to pixels as soons as they are drawn and does not remember what these pixels represent

There are three widely supported drawing styles
1. 2d - "for two dimensional graphics"
2. webgl - "for three dimensional graphics through the OpenGl interface
3. webgu - a more mordern and flexible alternative to webGl.
